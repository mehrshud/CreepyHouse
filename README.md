# CreepyHouse

## Introduction
CreepyHouse is a project that utilizes JavaScript and Node.js to create a spooky atmosphere.

## Usage Examples
### Installing Dependencies
To install the required dependencies, run the following command in your terminal:
```bash
npm install
```
### Running the Camera Controller
To run the camera controller, use the following command:
```javascript
node cameraController.js
```
### Example Use Cases
* Use the camera controller to capture spooky images or videos.
